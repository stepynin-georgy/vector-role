vector-role
=========

Download and install Vector

Role Variables
--------------

|vars           |descritption              |
|---------------|--------------------------|
|vector_workdir | Work direcory for Vector |
|vector_os_user | user                     |

Example Playbook
----------------

Including an example of how to use your role (for instance, with variables passed in as parameters) is always nice for users too:

    - hosts: servers
      roles:
